LIVE SITE

Book Club Reads is an app is lets the user search for books via an author or book title; for each search result, the app renders informations related to the book (think description, rating, genre, and more!) Users can use the MyBookshelf features to save books to their reading list and track the percentage of how many books they have completed on their list. This app uses the Google Books API, Firebase, has full English and French functionality, a predictive search function and was built with React. This app was produced by myself, Jonathan Lyn, Martin Nguyen and Brittany Barnard.
