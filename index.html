Book Club Reads

This app is lets the user search for books via an author or book title; for each search result, the app renders informations related to the book (think description, rating, genre, and more!) Users can use the MyBookshelf features to save books to their reading list and track the percentage of how many books they have completed on their list. This app uses the Google Books API, Firebase, has full English and French functionality, a predictive search function and was built with React.

Resto Recco PRO

Feeling hungry? This fun app lets the user specify their location and type of cuisine they're craving, then provides them with a list of restaurants that fits their search parameters. This app is powered by the Zomato API was built with Jquery.

The Tech Insider

Looking for the hottest news from the Tech Industry? Look no further. This app has a clean interface that allows the user to view the latest stories and also has a key word search function. Using the New York Times API, the user is provided with tiles containing the image, headline, time stamp, an abstract and a link to the full article.
